# Sorteador de Números

Projeto desenvolvido como parte dos estudos em JavaScript, com o objetivo de consolidar conceitos de lógica, manipulação do DOM e responsividade.

## Sobre o projeto

O sorteador de números permite ao usuário:

* Definir intervalo mínimo e máximo
* Escolher a quantidade de números
* Evitar números repetidos
* Visualizar os resultados de forma dinâmica

O layout foi baseado em um modelo do Figma, com algumas adaptações visuais feitas com objetivo de praticar as ferramentas.

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## Aprendizados

Durante o desenvolvimento, foram praticados:

* Manipulação do DOM
* Eventos no JavaScript
* Lógica de programação
* Responsividade com CSS
* Organização de layout
